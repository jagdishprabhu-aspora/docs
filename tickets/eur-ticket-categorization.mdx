## Overview

EUR ticket categorization follows the **exact same structure as UK**. The only difference is using "EUR →" prefix instead of "UK →".

<Info>
  All procedures, paths, and requirements are identical to UK. Simply replace "UK" with "EUR" in all ticket paths.
</Info>

## 1. EUR Verification

<Card title="EUR Verification Ticket" icon="id-card">
  **Ticket path:**
  ```
  EUR → Verification → Sumsub → User ID
  ```
  
  **Use when:**
  - Manual onboarding review required
  - Sumsub verification flagged user
  - Triggered during user onboarding
</Card>
